Sitios Recomendados

IMDB

IMDb, o Internet Movie Database, es una base de datos en línea que almacena información sobre películas, televisión, celebridades y videojuegos: Actores, Directores, Bandas sonoras, Series de televisión, Programas de televisión.

Filmaffinity

FilmAffinity es una página web que permite a los usuarios votar, recomendar y opinar sobre películas y series. Es una base de datos de cine en español con puntuaciones medias y críticas de usuarios. Permite a los usuarios crear listas de películas con características específicas.

Rotten Tomatoes

Rotten Tomatoes es un sitio web que recopila y analiza reseñas de películas y programas de televisión para ofrecer una puntuación llamada "Tomatómetro". Califica las películas y programas de televisión con base en las opiniones de críticos y audiencia. Las películas con más del 60% de reseñas positivas se consideran "frescas" y reciben un icono de tomate.

Metacritic

Metacritic es un sitio web que recopila reseñas y proporciona una nota media para películas, videojuegos, programas de televisión y álbumes de música. Metacritic calcula una nota media ponderada de las puntuaciones de las reseñas de cada producto. Para ello, selecciona a un grupo de críticos respetados y asigna una puntuación a cada una de sus reseñas. El resultado es un número y un color que se conoce como Metascore, el cual representa el consenso crítico para cada producto.

Sensacine

Sensacine ofrece a los usuarios la opción de registrarse, para puntuar y criticar películas, elegir sus cines preferidos y participar en concursos. Cada película tiene tres puntuaciones: la de SensaCine, una media de las puntuaciones dadas por los medios especializados y la media de las puntuaciones de los usuarios.
